# 调查主界面设计文档

## 设计目标

打造一个**叙事调查台**，而非传统后台或纯视觉小说界面：

- 玩家能快速理解"当前所处场景"和"正在发生什么"
- 玩家能便捷执行：前往地点 / 询问 NPC / 调查物品
- 玩家能立即感知：时间推进 / 风险变化 / 情报更新
- 玩家能方便进入：地图 / 线索册 / 侦探板 / 对话记录（但不抢主流程）

## 整体风格

### 视觉风格

- **主题**：哥特神秘风格
- **基调**：暗紫与黑色基调、华丽花纹、复古墨迹效果
- **配色方案（午夜酒红）**：
  - 背景：#140D12（暗红棕）
  - 主色/卡片背景：#261820（深棕红）
  - 强调色：#BE4BDB（酒红紫）
  - 文字主色：#F0EBE5（米白）
  - 文字次要：#9CA3AF（灰白）
  - 边框/分割线：#3D2829（暗棕）
  - 高亮/悬停：#D946EF（亮紫）
  - 危险/警告：#DC2626（暗红）

### 字体

- **标题**：Playfair Display（衬线体，强调文献感与侦探氛围）
- **正文**：Inter（无衬线，保证可读性）
- **打字机文本**：Courier Prime（用于日期、档案编号等）
- **基准字号**：16px，行高1.7

### 装饰风格

- **边框**：墨迹晕染效果 —— 边缘不规则的深色渐变，像墨水浸染纸张
- **背景**：可添加细微的纸张纹理或噪点效果
- **滚动条**：自定义暗色风格

## 页面布局

### 整体结构

采用 **CSS Grid 固定比例布局**：

```
+--------------------------------------------------+
|  [顶部信息栏：时间 | 暴露度 | 当前局势提示]       |  固定高度 60px
+----------+------------------------------+----------+
|          |                              |          |
|   20%    |            60%               |   20%    |
|  左侧    |        中央主舞台            |  右侧    |
|  行动列  |                              |  案情列  |
|          |                              |          |
+----------+------------------------------+----------+
|  [底部情报带：地图 | 线索册 | 侦探板 | 对话记录]   |  固定高度 50px
+--------------------------------------------------+
```

### 顶部信息栏（固定高度 60px）

**内容**：
- 左侧：当前时间（打字机风格：1924.10.15）
- 中间：当前局势小提示（一句话概括）
- 右侧：暴露度/风险指示器（裂纹蔓延效果）

**样式**：
- 背景略深于主背景
- 下边框带细墨迹效果

### 左侧行动列（20%宽度，可滚动）

**内容分组**：

1. **当前地点卡**
   - 地点名称
   - 简短描述
   - 天气/时间状态图标

2. **可前往地点**
   - 地点列表（3-6项）
   - 每个地点显示：名称、距离/耗时、是否解锁
   - 悬停：墨水扩散效果

3. **可询问NPC**
   - NPC列表（3-6项）
   - 每个NPC显示：剪影头像、姓名、身份标签
   - 当前对话中NPC高亮

4. **可调查物品**
   - 物品列表（3-6项）
   - 每个物品显示：图标/缩略图、名称、状态
   - 已调查物品标记

**样式**：
- 各分组用细线分隔
- 分组标题：小字大写，带装饰符号
- 列表项：卡片式，悬停时背景变亮

### 中央主舞台（60%宽度，独立滚动）

**三种状态**：

#### 1. 默认调查态

显示内容：
- 地点标题（大字Playfair Display）
- 场景描述（段落文本）
- 在场人物与可见物品摘要
- 当前局势小提示

#### 2. 对话进行态

显示内容：
- 对话对象名与身份标签
- NPC对话正文（带引号或对话框样式）
- 当前话题标签
- 可继续追问的话题选项（按钮列表）
- 时间消耗提示

**交互**：
- 话题按钮悬停：墨水扩散
- 选择后：新文本淡入

#### 3. 调查反馈态

显示内容：
- 被调查对象标题
- 调查结果正文
- 获得的线索卡（可点击翻转）
- 状态变化提示：
  - 时间推进
  - 解锁新地点
  - 新增可询问对象
  - 风险上升

**线索卡设计**：
- 正面：线索标题、摘要、图标
- 背面：详细信息、关联线索、备注栏
- 翻转动画：CSS 3D翻转
- 新线索出现时：简约脉冲效果

### 右侧案情列（20%宽度，可滚动）

**内容**：

1. **时间显示**
   - 完整日期：1924年10月15日
   - 时段：傍晚
   - 天气：雨

2. **暴露度/风险指示器**
   - 裂纹蔓延视觉效果
   - 低风险：边缘细微裂纹
   - 中风险：裂纹延伸至中间
   - 高风险：裂纹遍布，中心泛红

3. **当前在场NPC**
   - 剪影头像列表
   - 点击可快速切换对话

4. **最新线索摘要**
   - 最近3条线索标题
   - 点击查看详情

5. **最近事件提醒**
   - 时间线形式的简要事件

### 底部情报带（100%宽度，固定高度50px）

**入口**：
- 地图（带地点标记数量角标）
- 线索册（带未读线索角标）
- 侦探板（带可连接线索提示）
- 对话记录

**样式**：
- 低视觉重量，图标+文字
- 有更新时带角标或状态点
- 悬停：背景高亮

## 交互效果规范

### 按钮悬停

- **效果**：墨水扩散
- **实现**：伪元素圆形背景，从中心放大，透明度渐变
- **时长**：300ms ease-out

### 线索卡翻转

- **效果**：3D Y轴翻转
- **触发**：点击
- **时长**：500ms ease-in-out
- **背面**：显示详情

### 线索发现提示

- **效果**：简约脉冲
- **实现**：box-shadow脉冲动画 + 轻微scale变化
- **时长**：600ms，重复2次

### 风险指示器

- **效果**：裂纹蔓延
- **实现**：SVG裂纹路径，根据风险值控制stroke-dashoffset
- **颜色**：从灰白渐变为暗红

### 页面切换

- **效果**：淡入淡出
- **时长**：200ms
- **保持**：右侧案情列信息始终可见

## 技术实现

### 技术栈

- React 19 + TypeScript
- React Router v7
- TailwindCSS v4
- 字体：Google Fonts（Playfair Display, Inter, Courier Prime）

### 布局代码

```css
/* 主容器 */
.investigation-layout {
  display: grid;
  grid-template-rows: 60px 1fr 50px;
  height: 100vh;
  overflow: hidden;
}

/* 中间三栏 */
.main-content {
  display: grid;
  grid-template-columns: 20% 60% 20%;
  overflow: hidden;
}

/* 可滚动区域 */
.scrollable {
  overflow-y: auto;
  scrollbar-width: thin;
  scrollbar-color: #3D2829 #140D12;
}
```

### 组件清单

1. `InvestigationLayout` - 主布局容器
2. `TopBar` - 顶部信息栏
3. `ActionColumn` - 左侧行动列
4. `MainStage` - 中央主舞台
5. `CaseColumn` - 右侧案情列
6. `IntelBar` - 底部情报带
7. `LocationCard` - 地点卡片
8. `NPCList` - NPC列表
9. `ItemList` - 物品列表
10. `ClueCard` - 线索卡（可翻转）
11. `RiskIndicator` - 风险指示器
12. `TimeDisplay` - 时间显示
13. `DialogueView` - 对话视图
14. `InvestigationView` - 调查视图

## 响应式考虑

- **桌面端**：完整四区布局
- **中等屏幕**（<1200px）：左右栏宽度固定200px，中间自适应
- **暂不处理移动端**：首版专注桌面端

## 可访问性

- 颜色对比度符合WCAG AA标准
- 所有交互元素有清晰的焦点状态
- 支持键盘导航
- 减少动效选项（prefers-reduced-motion）
